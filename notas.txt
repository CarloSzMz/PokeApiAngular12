Set Up:
    Para iniciar el proyecto usar npm run start
    En el fichero package.json cambiar la linea de start por: "start": "SET NODE_OPTIONS=--openssl-legacy-provider && ng serve",

Configuracion:
    AppComponent tiene el headerComponent y footerComponent ya que se utilizan en todas las paginas.
    También contiene el Router-Outlet, que para que funcione hay que configurar el app-routing.module.ts
    Y también hay que incluir el import de AppRoutingModule en el archivo app.module.ts 

Componentes:
    -headerComponent contiene una imagen que lleva al inicio
        Tiene un buscador por nombre,que es un formulario que para que funcione necesita importar en app.module.ts el modulo FormsModule
        Este formulario llama a la ruta de detailComponent pasandole como parametro el nombre introducido en el form 

    -homeComponent contiene la pagina inicial del proyecto donde se muestran todos los resultados.
        Llama al servicio para utilizar sus metodos y recoger los datos para luego pasarselos a su HTML
        Contiene un boton para pasar al componente detailComponent donde se veran detalles del elemento escogido.
        
    -detailComponent muestra el elemento elegido en el homeComponent.
        Hay que declarar su ruta ya que va a cambiar el contenido de la pagina a una nueva en el archivo app-routing.module.ts
        En esta ruta se indica que se le pasa un parametro id del elemento que quiere ver el detalle
        Para ver ese id que esta en la ruta de la pagina se utiliza el modulo ActivatedRoute
        Este modulo se pone en el constructor y asi se saca el id de la ruta: const id = this.route.paramMap.subscribe((params) => {const id = params.get('id');})
        Y despues de recoger el id o nombre, se realiza la busqueda llamando al metodo del servicio
        Contiene botones de anterior y siguente. Llaman al metodo de buscar utilizando el id anterior/siguente del actual

Servicios:
    RestService: hace las llamadas a la API
    *Necesario importar el HttpClient en app.module.ts
